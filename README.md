# 🧠 Retail AI Platform

An end-to-end AI-powered platform for analyzing user behavior and predicting purchase intent in modern retail environments.

🔗 **Live Demo:**  
https://retail-ai-platform-m9rr.vercel.app/

---

## 🚀 Overview

Retail AI Platform is a full-stack application that simulates real-world retail analytics systems used by e-commerce companies.

It combines machine learning with modern web technologies to:

- Analyze user session behavior
- Visualize engagement and funnel metrics
- Predict purchase intent in real-time
- Generate actionable business insights

---

## 🧩 Features

### 📊 Advanced Analytics Dashboard
- Funnel analysis (view → click → add-to-cart → purchase)
- Conversion rate & drop-off tracking
- Channel, device, and region insights
- Interactive charts and visualizations
- Real-time data rendering

---

### 🤖 AI-Powered Prediction
- Predicts if a user will convert (purchase)
- Based on behavioral features such as:
  - Time spent
  - Interaction count
  - Clicks & views
  - Add-to-cart activity
  - Engagement score
- Instant prediction results via API
